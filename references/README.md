# Illustration Package — Reference Images

Side-by-side comparisons proving the illustration pipeline produces
visibly better output than raw `lineTo()` / `lineWidth` rendering.

## Structure

```
references/
├── phase-1-outlines/       ← Geometric test renders (Bars 1-6)
├── phase-2-junctions/      ← Junction test renders (Bars 7-11)
├── phase-3-marks/          ← Mark strategy renders (Bars 12-17)
├── phase-3-fills/          ← Fill strategy renders (Bars 18-20)
└── phase-4-integration/    ← Full integration renders (Bars 22-24, TODO)
```

## Convention

Each bar directory contains:
- `before.png` — Current rendering method (lineTo / lineWidth)
- `after.png` — Illustration pipeline output

Images are generated by `examples/render-bars.ts`.
